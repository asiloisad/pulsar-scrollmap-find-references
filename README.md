# scrollmap-find-references

Show find-references markers on the scrollbar.

When a language server highlights references to the symbol under the cursor, this package displays their positions as markers on the scrollmap scrollbar. It automatically disables the built-in scrollbar decoration from `pulsar-find-references` to avoid duplication.

## Installation

To install `scrollmap-find-references` search for [scrollmap-find-references](https://web.pulsar-edit.dev/packages/scrollmap-find-references) in the Install pane of the Pulsar settings or run `ppm install scrollmap-find-references`. Alternatively, you can run `ppm install asiloisad/pulsar-scrollmap-find-references` to install a package directly from the GitHub repository.

Layer package for [scrollmap](https://github.com/asiloisad/pulsar-scrollmap).

Requires [pulsar-find-references](https://github.com/savetheclocktower/pulsar-find-references).
